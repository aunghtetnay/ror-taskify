# Turbo (Hotwire)

## Built With

- Languages: Ruby
- Framework: Rails
- Database: SQLite

## Usage

Run the following commands:</br>
`git@github.com:aunghtetnay/taskify.git`</br>
`cd taskify`</br>
`bundle install`</br>
`bin/dev`</br>
Open a browser and go to: `https://localhost:3000` to view the app.
